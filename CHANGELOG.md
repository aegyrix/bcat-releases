# Changelog

All notable changes to [Bcat](https://bcat.app) are documented here.

## [v1.0.0] — 2026-04-12

### Bcat v1.0.0 — Initial Release

- Local AI chat powered by Ollama (Gemma 3, Llama 3.1)
- Multi-model switching with per-session model selection
- JWT-based authentication with secure login flow
- Conversation history with search and management
- Streaming responses with markdown rendering
- Dark UI with responsive mobile layout
- Self-hosted on personal VPS — no cloud dependencies
